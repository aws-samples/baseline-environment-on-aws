# Architecture Diagrams

## Rendering

The architecture diagram is defined in Mermaid format (`architecture.mmd`).

To render as PNG:

```bash
# Install mermaid-cli
npm install -g @mermaid-js/mermaid-cli

# Render
mmdc -i architecture.mmd -o architecture.png -t neutral -w 1200
```

Alternatively, paste the `.mmd` content into https://mermaid.live/ for browser-based rendering.
